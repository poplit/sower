■はじめに
　このドキュメントは手抜きです（こら）。

■設置方法（手抜き）
　perl5/CGI が動くUNIX系サーバならたいてい動くと思います（たぶん）。他のOS（WinやMac）はわかりません。

　各種設定は config.pl にあります。最低限 URL_SW の設定が必要です。

--------------------------------------------------
　config.pl を直接編集するとアップデート時にめんどくさい事になるので避けた方が無難です。_config_local.default を _config_local.pl にリネームした上で、対象となる設定部分を _config.pl からコピーしてから変更すると楽ができると思います。
※ベータ版はこの限りではありません（おい）。
--------------------------------------------------

　設定ファイルの変更に関しては同梱の config.txt にも目を通しておいて下さい。

　アップロードしたら、以下のようにすればたいてい動きます（たぶん）。
・/data 以下にある全てのファイル及びディレクトリに書き込み属性を与えて下さい。
・必要なら sow.cgi の先頭行にある perl のアドレスを、サーバの設定に合わせて下さい。
・sow.cgi に実行権限を与えて下さい。

--------------------------------------------------
※サーバの設定によっては、特別書き込み属性を与えなくても動きます。
※サーバの設定によっては _config_local.pl をいじる必要があるかもしれません。
※レンタルサーバを利用しているなら、詳しい事はサーバの管理者に尋ねてみて下さい。
--------------------------------------------------

　ちなみに sow.cgi の my $ENABLED_INSTALLCHECK = 0; という箇所を 1 に変えると、http://〜/sow.cgi?check でインストールチェックが行えます。タイトルバナーが表示されれば、インストールは出来ているはずです。たぶん。

　スクリプトが動いたら、admin と master でログインして下さい。この二つの ID は管理者用なので新規作成のためにログインする必要があります。

■設定上の注意点（手抜き）
　_info.pl.defalut は「管理人からのお知らせ」設定ファイルです。連絡事項などがあるならこのファイルを _info.pl にファイル名変更してから、適時書き直して下さい。
　禁止事項を変えたい時は rs/doc_prohibit.pl を書き換えて下さい。
　禁止事項以外にも、rs ディレクトリには各種説明書やキャラセット設定ファイルなどが収められています。いろいろいじってみたい方は覗いてみてください。

■アップデート方法（手抜き）
　基本的に丸ごと上書きすればアップデートできます。アップデートの手順は下記のようにすると良いでしょう。

　1._config_local.pl の ENABLED_HALT を 1 にして動作を止める
　2.ftpクライアントで上書きする
　3.ENABLED_HALT を 0 に戻して動作を再開

　禁止事項等を変更している時は rsディレクトリをそのまま上書きしないよう注意して下さい。
